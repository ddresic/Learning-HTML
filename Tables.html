An HTML table is defined with the <table> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered, but you can set a CSS "text-align" property to change that. A table data/cell is defined with the <td> tag. To add a table caption to a table, use <caption> tag.
Table will by defult be displayed without borders,a border is set using CSS "border" property, you need to define borders for both a table and a cells. If you want borders to colapse into one border,add the CSS "border-colapse" property and set value to "colapse",else every cell will display with its own border, you can also specify cpace between the cells borders by seting a CSS "border-spacing" property.
HTML table cells can span more then one row or column, to set this use "colspan" attribute for columns and "rowspan" for rows.
