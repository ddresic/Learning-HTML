Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is "block" or "inline."

Block-level Elements

A block level element always starts on new line and takes up the full width available (stratches out to left and right as far as it can).

One example of a block level element is the <div> element. The <div> element is often used as container for other HTML elements.
The <div> element requires no attributes,but "style", "class" and "id" are common for styling blocks of content.

Inline elements

An inline element does not start on a new line and only takes up as much width as necessary.Example this isan inline <span> element inside a paragraph.
