All HTML documents must start with a document type declaration <!DOCTYPE html>.
The HTML document itself begins with <html> and ends with </html>.
The <head> element is container for metadata,it is placed between <html> and <body> element,it is not displayed in a browser,but consists of a meta information such as document title, character set, styles, links, scripts, and other meta information.
The visible part of the HTML document is between <body> and </body>.
