CSS stands for Cascading Style Sheets.
CSS describes how HTML elements are displayed on screen,paper, or in other media.
CSS saves alot of work.It can control the layout of multiple web pages all at once.
CSS can be added to HTML in 3 weys

  • Inline - by using a style atribute in HTML elements
  • Internal - by usng a <style> element in <head> section
  • External - by using an external CSS file

An internal CSS is used to define a style for a single HTML page and it`s defined inside a head section of a HTML page,within a <style> element.
An external CSS can be written in any text editor,the file must not contain any HTML code and must be saved with a .CSS extension. Each page must include a reference to external CSS file within "link" element,inside "head" section.
The style of a text can be chenged by using "color" property to define a color of a text,"font-family" property to define which font to be used and "font-size" property.
The CSS "border" property defines a border aroundan HTML element and it has 3 values,width, style and color of a border.
The CSS "padding" property defines the space between text and border.
The CSS "margin" element defines a space outside the border.
"id" atribute is used to identify a unique element to which an style should apply.
"class" atribute is used to define a style for a specific class of elements.
