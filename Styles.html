Setting the style of an HTML element,can be done with the style attribute.
To add style attribute to the tag,write style="property:value; " inside opening tag,after the tag name.

< tagname Style="property:value; " >

The property is a CSS property and a value is a CSS value

The 'background-color' property defines the background color of an element, the 'color' property defines the color of a text,'font-family' defines font to be used,'text-size' the text size,'text-align'the horizontal alignment of text.
