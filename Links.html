HTML links are hyperlinks which allow user to jump to another document, by clicking on it.
In HTML links are defined with the <a> tag, the destination addres is specified with a "href" attribute, the "title" attribute specifies extra information about linked document, information will be shown when mouse hovers over the element you can also spcify were to open linked document with a "target" atribute.
Target attribute can have one of the following values:

  • "_blank" - opens document in a new window/tab.
  • "_self" - opens document in the same window/tab
  • "_parent" - opens document in the parent frame
  • "_top" - opens document in the full body of a window
  • framename - opens document in a named frame

HTML links can also be used to create a bookmarks,they allow users to jump to a specific part of page. To create a bookmark you first have to identify it with a "id" attribute,then add link to it, when the link is clicked, the page will automaticli scroll to the location with the bookmark. Bookmarks can be usefull for very long pages.
