In HTML, images are defined with the <img> tag. The <img> tag is rmpty tag, it contains only attributes, and does not require a closing tag.
The "alt" attribute provides an alternative text for an image, if for some reason user is unable to view an image,browser will display the value of an "alt" attribute. Style attribute is used to specify an height and width of an image
Alternativly, you can use the "width" and "height" attributes. However, it is sughested to use style attribute, because it prevents style sheets from changing the size of images.
Images can also be used as links, if contained within <a> tag.

The <map> tag defines an image-map,which is an image with clickable areas.

Images can also be used for an background. To add a background image on an HTML element, use the CSS property "background-image"

HTML5 introduced the <picture> element to add more flexibility when specifying images resources.
The <picture> element contains a number of <source> elements,each rederring to different image source. This way the browser can choose the image that best fits the current view and/or device. Each <source> element have attributes describing when their image is the most suitable. The browser will use the first <source> element with matching attribute values, and ignore any following <sorce> elements.
