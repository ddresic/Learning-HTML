Color Values

In HTML, colors can also be specified using rgb values,HEX values, HSL values, RGBA values and HSLA values:
Color "tomato" usind different values:

RGB(255, 99, 71)

RGB stands for RED, GREEN, BLUE. Each parametar defines the intesity of the color between 0 and 255.

HEX Value: #ff6347

HEX stands for hexadecimal and it uses the same principal as RGB,but the values in this system go from 00 to ff

HSL Value: hsl(9, 100%, 64%)

HSL stands for HUE, SATURATION, LIGHTNESS.
Hue is a degree on the color wheel,from 0 to 360, 0 is red, 120 is green, 240 is blue.
Saturation is a procentage value, 0% means shade of grey and 100% is a clear color.
Lightness is also a procentage, 0% is black, 50% is neither light nor dark, 100% is white.

RGBA Value: rgba(255, 99, 71, 0.5)

HSL Value: hsla(9, 100%, 64%, 0.5)

A at the end of both RGBA and HSLA stands for ALFA Parametar, it specifies opacity for a color and it`s value can go between 0.0 and 1.0, 0.0 being fully transparent and 1.0 not transparent at all
