Paragraphs

The HTML <p> element defines a paragraph.

With HTML, you cannot change the output by adding extra space or extra lines in your HTML code,because the browser will ignore any extra space or lines when the page is displayed.

Most browsers will display HTML correctly even if you forget the end tag <p>,but it can produce unexpected results or errors, so it shouldn`t be relyed on.

Use <br> element if you want a new line without starting a new paragraph.
The <br> tag is an empty tag,which means that it has no end tag.

The HTML <pre> element defines preformatted text.
The text inside a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks
