Unordered list

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag.By default list items will be marked with bullets, (small black circles), but this can be changed using the CSS "list-style-type" property,as shown below.

  • disc/default
  • square
  • circle
  • none

An ordered list starts with the <ol> tag. Each list item starts with the <li> tag. By default, an ordered lists will start counting from 1, if you want to start counting from specific number, you can use the "start" attribute. The list items will be marked with numbers by default , but this to can be changed using "type" attribute. As shown below

  1. type 1/default
  1. type A/uppercase letters
  1. type a/lowercase letters
  1. type I/uppercase roman numbers
  1. type i/lowercase roman numbers

HTML also supports description lists,which is a list of terms, with a description of each term. Description list is defined with the <dl> tag, term with <dt> tag and discription with the <dd> tag.

1. <dl> tag
- defines description list
2. <dt> tag
- defines term to be dscribed
3. <dd> tag
- defines description of the term

HTML lists can be styled in many different ways with CSS. One popular way is to style a list horizontally, to create a navigacion menu
